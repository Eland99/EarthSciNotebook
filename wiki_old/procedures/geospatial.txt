======Geospatial data======
This page is a placeholder for procedures that involve geospatial data. There are a few links. That is all.

=====Data products=====
  * [[http://prism.oregonstate.edu/|The PRISM group homepage]] with access to datasets (Oregon State Univ.).
  * [[http://www.wcc.nrcs.usda.gov/climate/prism.html|PRISM data]] - available at the NRCS website

=====Software=====
  * [[http://www.osgeo.org/|Open Source Geospation Foundation (OSGeo)]] - provides a portal to many open source geospatial software projects, like:
    * [[http://grass.osgeo.org/|GRASS desktop GIS]] - geospatial data management and analysis, image processing, graphics/maps production, spatial modeling, and visualization.
    * [[http://qgis.osgeo.org/|Qgis]] - a more user friendly desktop GIS that supports vector, raster, and database formats and offers integration with GRASS.
  * [[http://www.gdal.org/|GDAL libraries]] - translator library for many raster geospatial data formats (Geospatial Data Abstraction Library)
  * [[http://www.gdal.org/ogr/index.html|OGR libraries]] - translator library for simple features vector data (related to GDAL).
  * [[http://nco.sourceforge.net/|NCO]] - a collection of utilities that aid manipulation and analysis of gridded scientific data (like climate/circulation data).
